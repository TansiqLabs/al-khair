# Al-Khair - Donation Management System

A modern, lightweight donation and charity management system designed for organizations to track donors, manage projects, and generate comprehensive reports.

## Features

- **Donor Management**: Complete donor profiles with contact information and donation history
- **Project Tracking**: Monitor expenses, beneficiary information with photos and documents
- **Smart Reports**: Generate detailed reports for donations and project expenses
- **Role-Based Access**: Separate access levels for admins and staff
- **Auto-Update System**: One-click updates directly from dashboard
- **Modern Interface**: World-class, responsive design
- **Lightweight**: Optimized for shared hosting environments

## Installation

1. Upload all files to your hosting directory
2. Visit your domain in browser
3. Follow the installation wizard
4. Start managing donations!

## System Requirements

- PHP 7.4 or higher
- MySQL 5.7 or higher
- Apache/Nginx web server
- 50MB+ disk space

## Technology Stack

- Backend: PHP (vanilla, optimized)
- Database: MySQL
- Frontend: HTML5, CSS3, JavaScript (vanilla)
- No heavy frameworks - pure performance
